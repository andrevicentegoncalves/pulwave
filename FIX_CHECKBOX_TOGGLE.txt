// PreferencesSection was receiving onChange={handleChange} but checkboxes 
// need onCheckboxChange={handleCheckboxChange}.
// 
// handleChange expects e.target.value (for text inputs)
// handleCheckboxChange expects e.target.checked (for checkboxes)
//
// By adding the onCheckboxChange prop, the checkboxes will now toggle correctly!
