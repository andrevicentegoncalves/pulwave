# App Store Distribution

Getting your app into the hands of users.

## When to Use

- Preparing for initial release
- Handling app rejections
- Managing Beta groups (TestFlight/Internal Testing)
- Optimizing store listing (ASO)

## Quick Reference

### Launch Checklist
1. **Screenshots**: High res, all device sizes (6.5", 5.5", 12.9" iPad).
2. **Privacy Policy**: URL required for submission.
3. **Support URL**: URL required.
4. **Build Version**: Must be higher than previous upload (e.g., 1.0.1).
